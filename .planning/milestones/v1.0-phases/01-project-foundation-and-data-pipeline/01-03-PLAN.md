       - Implement partition_iid(X: np.ndarray, y: np.ndarray, num_clients: int, seed: int) -> list[tuple[np.ndarray, np.ndarray]]:
         a. Use sklearn.model_selection.StratifiedKFold(n_splits=num_clients, shuffle=True, random_state=seed)
         b. For each fold, use the TEST indices (not train) as the partition shard (StratifiedKFold gives train/test per fold -- we want the test portion as each client's data, since with K folds the K test portions are non-overlapping and cover all data)
         c. VALIDATION GATE: assert each partition class ratio within 5% of global
         d. Log per-partition: size, class distribution (count and percentage)
         e. Return list of (X_partition, y_partition) tuples

       - Implement create_dataloaders(partitions: list[tuple[np.ndarray, np.ndarray]], X_test: np.ndarray, y_test: np.ndarray, batch_size: int) -> tuple[list[DataLoader], DataLoader]:
         a. For each partition: create TensorDataset(torch.tensor(X, dtype=torch.float32), torch.tensor(y, dtype=torch.int64)), wrap in DataLoader(shuffle=True, batch_size=batch_size)
         b. Create test DataLoader similarly but with shuffle=False
         c. Return (list_of_client_loaders, test_loader)

       - Google-style docstrings. Use logging module.

    3. UPDATE src/federated_ids/data/preprocess.py -- add main() function and __main__ support:
       - Add main(config_path: str = None) -> tuple[list[DataLoader], DataLoader]:
         a. Parse CLI args (argparse): --config path/to/config.yaml (optional, defaults to config/default.yaml)
         b. Load config via load_config()
         c. Set global seed via set_global_seed()
         d. Check if processed data already exists in processed_dir (X_train.pt, X_test.pt, y_train.pt, y_test.pt). If yes, load them directly, skip loading + preprocessing. Log "Loading cached data from {processed_dir}".
         e. If not cached: call load_cicids2017() then preprocess(). Save processed tensors to processed_dir as .pt files (torch.save). Also saves scaler/features/weights as already done by preprocess().
         f. Call partition_iid() on X_train, y_train
         g. Call create_dataloaders() to wrap in DataLoaders
         h. Log summary: num clients, samples per client, test set size, device
         i. Return (client_loaders, test_loader)

       - Add if __name__ == "__main__": main() block
       - Create src/federated_ids/data/__main__.py for python -m federated_ids.data.preprocess support
