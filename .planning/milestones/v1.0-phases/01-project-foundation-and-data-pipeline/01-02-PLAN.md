    2. Create src/federated_ids/data/preprocess.py:
       - Implement select_features(df: pd.DataFrame, config: dict) -> tuple[pd.DataFrame, dict]:
         a. Drop IDENTIFIER_COLS (from loader.py constants, import them)
         b. Separate features (X) from Label (y)
         c. Domain shortlist: start with DDoS-relevant features from RESEARCH.md column list. Keep all columns that exist in the data from this shortlist. If fewer than target_features remain after domain filter, keep all non-identifier numeric columns instead (fallback).
         d. Drop zero-variance features (variance < config["data"]["variance_threshold"])
         e. Drop near-constant features (>99% same value)
         f. Compute correlation matrix, drop one from each pair with |r| > config["data"]["correlation_threshold"] (keep the one with higher variance)
         g. Build feature_report: dict mapping dropped feature names to drop reasons
         h. Log the report (each dropped feature + reason, final feature count)
         i. Return (filtered DataFrame with Label column reattached, feature_report)

       - Implement preprocess(df: pd.DataFrame, config: dict) -> dict:
         a. Call select_features to get filtered df and report
         b. Extract X (feature columns as float32 ndarray), y (Label as int64 ndarray)
         c. Global train/test split using sklearn.model_selection.train_test_split(stratify=y, test_size=config["data"]["test_size"], random_state=config["seed"])
         d. Fit StandardScaler on X_train ONLY. Transform X_train (fit_transform) and X_test (transform only). Cast back to float32 after scaling.
         e. Compute class weights: sklearn.utils.class_weight.compute_class_weight("balanced", classes=np.unique(y_train), y=y_train). Convert to dict {0: weight0, 1: weight1}.
         f. Save artifacts to config["data"]["processed_dir"]:
            - Fitted scaler: joblib.dump(scaler, "scaler.pkl")
            - Feature list: json.dump(list of selected feature names, "features.json")
            - Class weights: json.dump(class_weights_dict, "class_weights.json")
            - Class distribution stats: json.dump with train/test counts per class
         g. Log: feature count, train/test sizes, class distribution per split, class weights
         h. VALIDATION GATES: no NaN in X_train/X_test, no Inf, correct dtypes
         i. Return dict: {"X_train": X_train, "X_test": X_test, "y_train": y_train, "y_test": y_test, "scaler": scaler, "feature_names": feature_names, "class_weights": class_weights_dict, "feature_report": feature_report}

       - Google-style docstrings on module and all functions.
       - Use logging module consistently.
  </action>
