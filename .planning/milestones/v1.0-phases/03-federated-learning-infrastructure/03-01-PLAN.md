  <name>Task 2: FederatedClient class and FedAvg aggregation</name>
  <files>src/federated_ids/fl/client.py, src/federated_ids/fl/server.py, src/federated_ids/fl/__init__.py</files>
  <behavior>
    - FederatedClient.get_parameters() returns list[np.ndarray] from model.state_dict()
    - FederatedClient.set_parameters() loads list[np.ndarray] into model via OrderedDict
    - FederatedClient.fit() calls set_parameters, runs train_one_epoch for local_epochs iterations, returns (get_parameters(), num_examples, {})
    - fedavg_aggregate() computes weighted average: for each layer, sum(param_i * n_i / total_n)
    - server_evaluate() loads global params into model, calls evaluate(), returns metrics dict
  </behavior>
  <action>
1. Create `src/federated_ids/fl/client.py`:
   - Class `FederatedClient` with `__init__(self, model, train_loader, criterion, optimizer, device)`.
   - `get_parameters() -> list[np.ndarray]`: `[val.cpu().numpy() for _, val in self.model.state_dict().items()]`
   - `set_parameters(parameters: list[np.ndarray]) -> None`: Build `OrderedDict` from `zip(self.model.state_dict().keys(), parameters)`, using `torch.tensor(v)` for each, then `model.load_state_dict(state_dict, strict=True)`.
   - `fit(parameters: list[np.ndarray], config: dict) -> tuple[list[np.ndarray], int, dict]`: Call `set_parameters(parameters)`, loop `config.get("local_epochs", 1)` times calling `train_one_epoch(self.model, self.train_loader, self.criterion, self.optimizer, self.device)`, return `(self.get_parameters(), len(self.train_loader.dataset), {})`.
   - Google-style docstring on class and all methods. Interface mirrors Flower NumPyClient for future migration (document in class docstring).
   - Import `from collections import OrderedDict`, `numpy as np`, `torch`, and from project: `train_one_epoch` from `federated_ids.model.train`.
   - Use module-level `logger = logging.getLogger(__name__)`.

2. Create `src/federated_ids/fl/server.py`:
   - `fedavg_aggregate(results: list[tuple[list[np.ndarray], int]]) -> list[np.ndarray]`: Compute `total_examples = sum(n for _, n in results)`. Initialize aggregated as `[np.zeros_like(results[0][0][i]) for i in range(num_layers)]`. Loop over results, `weight = num_examples / total_examples`, accumulate `aggregated[i] += param * weight`. Return aggregated.
   - `server_evaluate(global_params: list[np.ndarray], model: torch.nn.Module, test_loader, criterion, device) -> dict`: Build `OrderedDict` from `zip(model.state_dict().keys(), global_params)` with `torch.tensor(v).to(device)`, call `model.load_state_dict(state_dict, strict=True)`, return `evaluate(model, test_loader, criterion, device)`.
   - Google-style docstrings. Module-level logger.

3. Update `src/federated_ids/fl/__init__.py`:
   - Add module docstring: "Federated learning infrastructure with pure-Python FedAvg."
   - Add `from federated_ids.fl.client import FederatedClient` and `from federated_ids.fl.server import fedavg_aggregate, server_evaluate`.
   - Define `__all__ = ["FederatedClient", "fedavg_aggregate", "server_evaluate"]`.

4. Run `pytest tests/test_fl.py -x -v` -- all tests MUST pass (GREEN).
5. Run `pytest tests/ -v` -- full suite must stay green (no regressions).
