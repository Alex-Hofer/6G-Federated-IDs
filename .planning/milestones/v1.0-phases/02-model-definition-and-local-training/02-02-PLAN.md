**src/federated_ids/model/train.py:**
Google-style module docstring. Import logging, os, json, argparse, torch, sklearn.metrics, sklearn.model_selection.train_test_split, and project modules (config, seed, device, MLP, partition_iid, data preprocess).

Implement these functions:

1. `train_one_epoch(model, train_loader, criterion, optimizer, device) -> float`:
   - model.train()
   - Iterate batches: zero_grad, forward, loss, backward, step
   - Accumulate total_loss weighted by batch size
   - Return average loss (total_loss / len(train_loader.dataset))

2. `evaluate(model, data_loader, criterion, device) -> dict`:
   - model.eval()
   - with torch.no_grad(): iterate batches, collect predictions and labels
   - Compute metrics using sklearn: accuracy_score, f1_score(average="binary"), precision_score(average="binary", zero_division=0), recall_score(average="binary", zero_division=0)
   - Return dict with keys: loss, accuracy, f1, precision, recall

3. `_print_summary_table(history: list[dict]) -> None`:
   - Log a thesis-appendix-ready table with columns: Epoch, Loss, Acc, F1, Prec, Rec
   - Use fixed-width formatting as shown in RESEARCH.md
   - Wrap with "--- Training Summary ---" header/footer

4. `standalone_train(config_path: str | None = None, client_id: int | None = None) -> None`:
   - Parse CLI args if config_path is None: --config (default config/default.yaml), --client-id (default 0)
   - If config_path provided programmatically, use it (skip argparse). If client_id is None, default to 0.
   - Load config, set seed, get device, set up logging
   - Check for cached tensors using _cache_exists. If missing, log and auto-run data pipeline: `from federated_ids.data.preprocess import main as run_pipeline; run_pipeline(config_path)`
   - Load cached tensors (X_train.pt, y_train.pt, X_test.pt, y_test.pt) with weights_only=True, convert to numpy
   - Call partition_iid() to get client partitions, select client_id
   - Stratified train/val split using sklearn train_test_split(stratify=y_client, test_size=val_split, random_state=seed)
   - Create DataLoaders for train, val, and test (TensorDataset wrapping float32/int64 tensors)
   - Instantiate MLP with input_dim from X_train.shape[1], hidden_layers/num_classes/dropout from config
   - Create Adam optimizer with config learning_rate
   - Load class_weights.json from processed_dir, convert to tensor on device, create CrossEntropyLoss(weight=weight_tensor). CRITICAL: weight tensor MUST be on same device as model.
   - Training loop for standalone_epochs:
     - train_loss = train_one_epoch(...)
     - val_metrics = evaluate(model, val_loader, ...)
     - Log: `Epoch {i}/{total} -- loss: {val_loss:.3f}, acc: {acc:.2f}, F1: {f1:.2f}, prec: {prec:.2f}, rec: {rec:.2f}`
     - If val F1 > best_f1: save state_dict to output_dir/checkpoints/best_model.pt, log `Saved best model (epoch {i}, F1={f1:.2f})`
     - Append val_metrics to history
   - Call _print_summary_table(history)
   - Final global test set evaluation: evaluate(model, test_loader, ...) and log with "--- Global Test Set Evaluation ---" header
   - Log format per user decision: `Epoch 3/5 -- loss: 0.142, acc: 0.95, F1: 0.83, prec: 0.87, rec: 0.79`
