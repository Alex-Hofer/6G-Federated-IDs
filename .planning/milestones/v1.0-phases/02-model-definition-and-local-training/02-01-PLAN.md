**src/federated_ids/model/model.py:**
- Google-style module docstring explaining this is the MLP for binary DDoS detection
- `class MLP(nn.Module)` with constructor args: `input_dim: int, hidden_layers: list[int], num_classes: int, dropout: float`
- Build layers dynamically from hidden_layers list: for each hidden dim, append nn.Linear, nn.ReLU, nn.Dropout(dropout)
- Final layer: nn.Linear(last_hidden, num_classes) -- NO softmax, NO activation
- Use nn.Sequential to store the layer stack in self.network
- forward(x) returns self.network(x) -- raw logits

CRITICAL: forward() must NOT apply softmax. CrossEntropyLoss applies log_softmax internally. Double-softmax is a common bug that prevents learning.
